## 📖 About

Lumina Guard is a comprehensive women's safety platform with real-time SOS alerts, intelligent location tracking, and instant emergency notifications. Designed for global use with modern glass morphism UI and professional branding.

**Internship Project:** Dilwado Web Development Internship (3 Months)

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas)

### Installation

1. Clone the repository
```bash
git clone https://github.com/YOUR-USERNAME/lumina-guard-safety-app.git
cd lumina-guard-safety-app
```

2. Install backend dependencies
```bash
cd backend
npm install
```

3. Install frontend dependencies
```bash
cd ../frontend
npm install
```

4. Configure environment variables

**Backend (.env):**
```env

```

**Frontend (.env):**
```env

```

5. Run the application

**Terminal 1 - Backend:**
```bash
cd backend
npm run dev
```

**Terminal 2 - Frontend:**
```bash
cd frontend
npm run dev
```

6. Open browser: `http://localhost:5173`
